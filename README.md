# org.buffalo-coders:jmh-logging

Java Microbenchmark Harness for `System.err`, `System.out`, and a variety of
popular logging frameworks.

## Usage

```
bash$ mvn --activate-profiles bundle
bash$ java -jar target/jmh-logging*.jar
```
